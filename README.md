# Thennavan Digital

Personal portfolio of Thennavan — Flutter developer and cybersecurity enthusiast building secure, scalable mobile apps.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Router DOM

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
